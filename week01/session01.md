# Week 01 – Session 01  
## C99 → C++ 사고방식 리마인드

---

## 이번 회차의 목적
- C99 기반 사고를 C++ 스타일로 교정한다
- 이후 자료구조 구현에서 사용할 개념 기준을 맞춘다
- “문법 복습”이 아니라 “개념 정렬”에 집중한다

---

## 핵심 전제
- C++은 C99의 상위 버전이 아니다
- 두 언어는 **사고방식이 다르다**

| 관점 | C99 | C++ |
|---|---|---|
| 변수 | 메모리 | 객체 |
| struct | 데이터 묶음 | 객체 |
| 할당 | 공간 확보 | 객체 생성 |
| 해제 | 메모리 해제 | 객체 소멸 |
